# What I learned from MSE 245 Lecture 1
- An interesting fact I learned from my first MSE 245 lecture was the fact that there were multiple types of "value absence" in JavaScript. In Python, the only value used to describe a statement missing a value is None, whereas in JavaScript, there are `null` and `undefined`. `Undefined` means that a value has yet to be set. Contrarily, `null` means that a value was set to intentionally be empty in the code.
